# Contributing to INFRA DRISHTI

Thank you for your interest in contributing to **INFRA DRISHTI**!  
We welcome contributions that improve the platform and help us move closer to our mission of building **safer public infrastructure through predictive technology**.

By contributing, you help make this project better for developers, researchers, and communities.

---

## Ways to Contribute

You can contribute in several ways:

- 🐛 Reporting bugs
- 💡 Suggesting new features
- 📖 Improving documentation
- 🧑‍💻 Submitting code improvements
- 🧪 Testing features and reporting feedback
- 🎨 Improving UI/UX design

---
